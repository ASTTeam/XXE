# 《深入理解XXE漏洞》

本项目用来收集整理XXE漏洞的相关内容，包括XXE的利用方法工具或思路等。XXE漏洞往往不可以执行命令，但是可以通过文件读取方法获取敏感信息，甚至进一步Getshell！作者：[ASTTeam](https://github.com/ASTTeam/XXE)

本项目创建于2022年3月3日，最近的一次更新时间为2022年3月6日。本项目会持续更新，直到海枯石烂！
